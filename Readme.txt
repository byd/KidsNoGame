
*******请牢记你所设的密码，不然会很麻烦********

1、程序打开后会自动隐藏

2、快捷键【Alt+D】打开或隐藏窗口

3、首次使用请设置密码，务必牢记密码，否则会比较麻烦

4、设置【开机启动】能发挥程序的最大功效

5、彻底删除程序请点击【卸载程序】

6、为保护本程序，请将"注册表"、"启动项"、"任务管理器"等关键字也添加进去。

===============================>

*适用于防止孩子玩游戏，或禁止别人在自己电脑上打开某类程序。

*程序每隔一秒钟扫描一次所有窗口，凡是标题带有关键字的窗口都会被关闭。

*程序只有一个执行文件，下载后将KillGames.exe拷贝到一个单独的目录里运行


说明：
1、为保证本程序不被破解，请将注册表编辑工具和进程管理工具也添加到监控列表中
  （我正在研究进程隐藏和防杀方法，有点和病毒类似了，有懂得的大虾还望指教。）
2、关闭窗口相当于直接点关闭按钮，不会对数据造成损失，也不会记录任何信息。
3、点击【卸载程序】会删除本程序的所有痕迹，再在弹出的窗口中把程序文件删掉即可，俺不流氓。
